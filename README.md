# Doce Encanto - Landing Page

Landing page moderna e responsiva para loja de doces artesanais, desenvolvida com foco em conversão e experiência do usuário.

## ✨ Características

- 🎯 Design moderno e atraente focado em conversão
- 📱 Totalmente responsivo (Mobile, Tablet, Desktop)
- ⚡ Performance otimizada e carregamento rápido
- 🎬 Animações suaves e interativas
- ♿ Acessível (WCAG 2.1)
- 💬 Botão WhatsApp flutuante para pedidos
- 🖼️ Imagens de alta qualidade
- 🎨 Paleta de cores vibrante e profissional

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Flexbox, Grid, Animations, Custom Properties)
- JavaScript Vanilla (ES6+)
- Intersection Observer API
- Google Fonts (Poppins, Playfair Display)

## 📋 Seções

1. **Hero** - Introdução impactante com CTA
2. **Sobre** - História e valores da empresa
3. **Produtos** - Catálogo com 6 categorias
4. **CTA Banner** - Call-to-action para conversão
5. **Depoimentos** - Avaliações de clientes
6. **Contato** - Informações de contato e localização
7. **Footer** - Links e redes sociais

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/tofariasti/lojadoces-landing.git
```

2. Abra o arquivo `site/index.html` no navegador ou:
```bash
cd lojadoces-landing
python3 -m http.server 8000
```

3. Acesse `http://localhost:8000/site/`
